# interface-feel-polish

Polishes the small interface details that make a frontend feel more readable, responsive, and intentional without redesigning it from scratch.

## Install

```bash
npx skills add https://github.com/Popwers/skills --skill interface-feel-polish
```

## What It Covers

- balanced text wrapping for short headings
- concentric border radius for nested surfaces
- contextual icon animation
- font smoothing for cleaner text rendering
- tabular numerals for stable metrics
- interruptible interactive transitions
- split and staggered enter animations
- softer exit animations
- optical alignment fixes
- layered shadows instead of harsh borders
- subtle image outlines

## Good Fits

- "Review this interface and make it feel more polished."
- "Refine the hover, loading, and exit states of this component."
- "Audit this dashboard for small UI details that feel off."
- "Improve typography, spacing, motion, and surface treatment without redesigning the page."
